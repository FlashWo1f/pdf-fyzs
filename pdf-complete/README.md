## 整体页面

### 静态页面

1. 目录页
2. 报告阅读指南
3. 报告维度
4. 评估维度 + 分数解读
5. 每个大维度的首页
6. 胜任力匹配
7. 封底

### 半静态页面 （需要数据填充）

1. 封面
2. 作答风格 ( 尽责 诚实  得分 )

### 动态页面

1. 整体得分情况的 Echarts 表
2. 每个维度的具体得分情况
3. 个人发展建议
4. HR 管理建议
